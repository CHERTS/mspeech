Список изменений в программе MSpeech

v1.5.10 от 15.02.2020:
* Исправлена ошибка, приводящая к краху программы, при нажатии горячей клавиши несколько раз подряд;
* Обновлены все компоненты MSpech до последних версий (MGSoft, RHVoice);
* Добавлен Украинский язык для распознавания;

v1.5.9 от 31.03.2016:
* Исправлена работа на Windows XP;
* Исправлен синтез речи через Google TTS;
* Мелкие исправления на странице добавления новых команд;

v1.5.8 от 02.03.2016:
* Исправлена потеря фокуса программы в которую MSpeech передает распознанный текст.

v1.5.7 от 27.09.2015:
+ Добавлены функции синтеза речи через онлайн-сервисы: iSpeechTTS и NuanceTTS;
+ Улучшен синтез речи через SAPI, для бесплатного синтеза речи используется RHVoice, доступны русский и английский языки;
+ Переход на использование собственных компонентов:
  - Синтез речи (MGSAPI, MGGoogleTTS, MGYandexTTS, MGiSpeechTTS, MGNuanceTTS);
  - Горячие клавиши (MGHotKeyManager);
  - Распознаванию текста (ASR);
  - Сохранение позиций форм (MGFormManager);
  - Группа кнопок (MGButtonGroup);
+ Отказ от использования компонентов JEDI и CoolTrayIcon, переход на использование собственных компонентов:
  - Потоковая загрузка больших лог-файлов (TMGTextReaderA, TMGTextReaderW);
  - Выполнение процедур в потоке (TMGThread);
  - Сворачивание программы в систрей (TMGTrayIcon);
+ Добавлен новый тип команды: Запустить программу с параметрами.
  Данная команда запускает указанную программу и передает ей распознанную строку в качестве 1 аргумента (см. в качестве примера script\Params.cmd).
* В просмотрщике лог-файлов устранена утечка памяти.

v1.5.6 от 11.07.2014:
+ Добавлены настройки: "Остановить распознавание после блокировки ПК" и "Запустить распознавание после разблокировки ПК"
  1. По-умолчанию после блокировки ПК происходит остановка записи и распознавания.
  2. По-умолчанию после разблокировки ПК запускается запись.
  Новые настройки расположены в категории "Настройки распознавания".
+ Добавлена возможность использования одинаковых комманд с разными типами.
  Например, добавим в список 2 новых команды:
  1. Ключевое слово: "запусти блокнот", Команда: "запускаю программу блокнот", Тип: "TextToSpeech"
  2. Ключевое слово: "запусти блокнот", Команда: "notepad.exe", Тип: "Запустить программу"
  После этого, как только Вы дадите MSpeech команду "запусти блокнот", вначале будет выполнена команда 1 и произнесена
  фраза "запускаю программу блокнот" через систему синтеза речи, а потом запущена программа по команде 2, то есть notepad.exe
+ Добавлена настройка: "Запускать MSpeech при входе в систему".
+ Добавлены новые языки для синтезатора речи Google TTS: Арабский, Датский, Немецкий, Греческий, Испанский,
  Финский, Французский, Итальянский, Японский, Корейский, Нидерландский (Голландский), Польский, Португальский, Турецкий, Китайский.
+ Добавлен синтезатор речи Yandex TTS. Поддерживаются следующие языки: Английский, Немецкий, Французский, Русский.
+ Добавлен синтезатор речи RHVoice. (RHVoice - это многоязычный синтезатор речи с открытым исходным кодом, доступный в версиях для
  операционных систем Windows и Linux. Разработчик синтезатора — Ольга Яковлева.) Для RHVoice доступны несколько голосов на русском и английском языке.

v1.5.5 от 19.05.2014:
+ Добавлена возможность автоматического получения Google API Key с сервера программы MSpeech.
  Данная функция включается и запрашивает новый ключ автоматически в случае, если произойдет ошибка распознавания
  по причине блокировки публичного API-ключа или Вашего личного API-ключа.
* Мелкие исправления в коде.

v1.5.4 от 14.05.2014:
+ Информация "О программе" перенесена в настройки.
+ Добавлена возможность активации доп. функций через Настройки -> "О программе".
+ Добавлена возможность ввода ключа для Google Speech API через интерфейс Настройки -> Настройки распознавания.
  Если данный ключ не введен, будет использован публичный ключ, но он может не работать, тогда будет выдаваться ошибка "Неизвестная ошибка в потоке распознавания".<br><br>

* Исправлена функция вызова настроек микрофона.
* Мелкие исправления в коде.
- Отключена фильтрация звука по различным алгоритмам.

v1.5.2 от 14.03.2014:

v1.5.1 от 17.11.2013:
+ Добавлен иврит.
* Исправлена проблема при отправке запроса на распознавание в Google.
* Добавлен второй язык распознавания как не основной и добавлена горячая клавиша для переключения языков распознавания.

v1.5.0 от 30.09.2013:
* Все настройки вынесены в отдельный интерфейс.
* Все файлы конфигурации перенесены в каталог профиля пользователя.
* При запуске cmd и bat-файлы выполняются как скрытые (SW_HIDE), остальные программы выполняются в нормальном режиме (SW_SHOWNORMAL).
+ Добавлено преобразование текста в речь (Microsoft SAPI или Google Text-To-Speech) для определенных видов событий.
+ Добавлен новый тип команд (TextToSpeech). Данный тип будет работать только если активировано преобразование текста в речь.
+ Добавлен интерфейс просмотра логов.
+ Добавлена горячая клавиша для начала записи и передачи текста в программы без выполнения управляющих команд.
+ Добавлена возможность отключения ведения лога операций в программе.
+ Добавлена фильтрация звука по различным алгоритмам (Sinc Filter - Фильтрация по диапозону частот (LowFreq, HighFreq), Voice Filter - Функции Automatic gain control (AGC), Noise Reduction, Voice Activity Detection (VAD))
+ Добавлена команда по-умолчанию, которая выполняется если не найдена ни одна команда из списка.

+ Добавлена кнопку выбора программы на вкладке "Команды".

* Файл *.flac пишется во временную директорию пользователя.

+ Добавлена возможность выбирать язык распознавания. Доступно более 50 языков распознавания.
+ Добавлен интерфейс редактирования команд.
